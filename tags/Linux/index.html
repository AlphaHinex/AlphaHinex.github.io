  • Linux

scp 是 Linux 上两个网络主机之间传输文件的命令行工具。

scp [options] source ... target
  • options 有很多，可通过 man scp 查看
  • source 为传输的源文件或路径，可以多个
  • target 为传输目标路径

source 或 target 为本地路径时，直接使用文件系统绝对路径即可；当为远程路径时，可使用如下两种格式：

  1. user@host:/path/to/file
